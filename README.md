# Enterprise Resource Controller

A distributed Java application built using the Model-View-Controller (MVC) architecture. This system demonstrates the implementation of remote service invocation and object persistence.

## Technical Highlights
* **Distributed Computing:** Implemented **Remote Method Invocation (RMI)** to allow client-server communication across a network.
* **Design Patterns:** Adheres strictly to the **MVC pattern**, ensuring a clean separation between the GUI, the business logic, and the data model.
* **Data Serialization:** Utilizes Java's native **Serialization** to preserve object state between sessions, ensuring data continuity.

## Tech Stack
* **Language:** Java
* **Network:** RMI
* **Pattern:** MVC
